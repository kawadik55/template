Список команд:
` /EditText` - редактировать выводимый кнопкой текст;
` /EditButtonName староеимя=новоеимя` - редактировать имя кнопки;
` /EditBackName новоеимя` - редактировать имя кнопки "Назад";
` /EditButtonUrl имякнопки=новыйURL` - изменить ссылку в кнопке;
` /EditButtonEg имякнопки=путькфайлу` - изменить путь к файлу Ежедневника;
` /EditButtonRaspis имякнопки=путькфайлу` - изменить путь к файлу Расписания;

` /AddFile имякнопки` - добавить документ(файл) в кнопку 'имякнопки';
` /AddPhoto имякнопки` - добавить фотку в кнопку 'имякнопки';
` /AddVideo имякнопки` - добавить видео в кнопку 'имякнопки';
` /AddAudio имякнопки` - добавить аудио в кнопку 'имякнопки';
` /AddHistory` - добавить Личную Историю;
` /AddButtonText имякнопки` - добавить кнопку типа 'текст';
` /AddButtonAdmin имякнопки` - добавить скрытую кнопку типа 'админ', видимую только Админам;
` /AddButtonTen имякнопки` - добавить кнопку типа '10шаг';
` /AddButtonUrl имякнопки=ссылка` - добавить кнопку типа 'ссылка';
` /AddButtonFile имякнопки` - добавить кнопку типа 'документ';
` /AddButtonPhoto имякнопки` - добавить кнопку типа 'фотки';
` /AddButtonVideo имякнопки` - добавить кнопку типа 'видео';
` /AddButtonAudio имякнопки` - добавить кнопку типа 'аудио';
` /AddButtonHistory имякнопки` - добавить кнопку типа 'истории';
` /AddButtonEg имякнопки[=путькфайлу]` - добавить кнопку типа 'ежедневник', \[ ]-необязательный параметр;
` /AddButtonRaspis имякнопки[=путькфайлу]` - добавить кнопку типа 'расписание', \[ ]-необязательный параметр;
` /AddButtonTime имякнопки` - добавить кнопку типа 'счетчик ЧВ';
` /AddButtonQuestions имякнопки` - добавить кнопку типа 'загрузить вопросы 10го шага';
` /AddButtonLocation имякнопки` - добавить кнопку запроса локации пользователя для определения часового пояса;
` /AddAdmin chatId=имяадмина` - добавить Админа в список;
` /AddAdmin show` - показать список Админов;
` /AddUser chatId=имяслуженца` - добавить служащего для бота;
` /AddUser show` - показать список служащих бота;

` /DelFile имякнопки` - удалить документ(файл) из-под кнопки 'имякнопки';
` /DelPhoto имякнопки` - удалить фотку из-под кнопки 'имякнопки';
` /DelVideo имякнопки` - удалить видео из-под кнопки 'имякнопки';
` /DelAudio имякнопки` - удалить аудио из-под кнопки 'имякнопки';
` /DelHistory` - удалить Личную Историю;
` /DelButton имякнопки` - удалить кнопку со всеми ветками;
` /DelAdmin chatId` - удалить Админа;
` /DelUser chatId` - удалить служащего бота;

` /DeadUsers` - удалить 'мертвых' подписчиков;
` /SignOff=1` - останавливает подписку новых юзеров, =0 - возобновляет;

` /MoveButtonUp имякнопки` - переместить кнопку вверх;
` /MoveButtonDown имякнопки` - переместить кнопку вниз;
` /CutButton` имякнопки - вырезать кнопку со всеми ветками;
` /InsertButton имякнопки` - вставить кнопку со всеми ветками;

` /StatWeek` - вывести статистику посещений за 2 недели;
` /StatGrand` - вывести статистику посещений за все время;
` /PublicText` - послать важное сообщение всем подписчикам;
` /PublicTextAdmin` - послать важное сообщение всем Админам и Юзерам;
` /PublicMessUser chatId=текстСообщения` - послать важное сообщение конкретному подписчику;
` /help` - вывести список команд;
