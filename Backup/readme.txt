Здесь находятся примеры скриптов для резервного копирования данных.
Скрипты запускаются в назначенное время из службы crontab.
Файл crontab.bkp показывает примерный список заданий в crontab.

Скрипт webdav_vk.sh показывает пример копирования нужных папок в облачное хранилище mail.ru.
Для его работы необходимо произвести предварительную настройку, этот процесс описан в файле WebDav.txt
С помощью службы davfs удаленный диск монтируется в настроенную папку, и в эту папку копируются нужные файлы стандартными средствами.
По окончании копирования удаленный диск отключается от системы.

Конечно же, все пути нужно исправить на свои.
